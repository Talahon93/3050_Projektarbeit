# 3050_Projektarbeit

# Fokusfrage: Das Wetter Beeinflusst die Passantenzahl?

## Diese Web-App soll die Fokusfrage beantworten und die Daten des "Gesamtdatensatz.csv" visualisieren. Dieser Datensatz beinhaltet Daten zu den Fussgängern an vier verschiedenene Zugängen zum Hauptbahnhof in der Stadt Zürich.

# Starten der Web-App

## Für die Benutzung diese Web-App werden Folgende Module benötigt:

### Frontend: React, Vite

### Backend: pathlib, pandas, fastapi

## Die Benutzung unserer Web-App kann starten sobald die oben aufgeführten Module Installiert wurden.

## 1. Das Richtige Enviroment wählen in welchem die Modlue Installiert wurden.

## 2. Zum Backend Ordner in der Vscode Konsole navigieren mit -> cd backend

## 3. Das Backend Starten mit -> fastapi dev main.py

## 4. Im Browser das Backend auf Funktionalität prüfen mit -> http://127.0.0.1:8000

## 5. In der Konsole von Vscode Zum Frontend navigieren mit -> cd .. und anschliessend mit -> cd frontend

## 6. Das Frontend Starten mit -> npm run dev

## 7. Im Browser die Web-App öffnen mit -> http://localhost:5173/

# Benutzen der Web-App

## In der Web-App giebt es Drei Seiten: Startseite, Fokusfrage und Exploration.

## Startseite: Giebt Aufschluss zur Fokusfrage und beschreibt die Web-App.

## Fokusfrage: Behnadelt die Fokusfrage und giebt Aufschluss zu den Erkentnissen.

## Exploration: Ermöglicht es dem Web-App benutzer die Daten eigenhändig über zeiträume und Standorte zu filtrieren und Darzustellen.
